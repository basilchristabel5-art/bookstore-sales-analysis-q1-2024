# bookstore-sales-analysis-q1-2024
Data visualization project analyzing Q1 2024 bookstore sales using Tableau. Features dashboards, insights, and recommendations focused on genre performance, regional trends, publisher impact, and customer order patterns.

Q1 2024 Bookstore Sales Analysis (Tableau Project):
This project presents a complete Tableau analysis of bookstore performance for Q1 2024. It explores genre trends, regional strengths, weekly sales patterns, publisher contributions, and customer order behavior. The goal is to identify key revenue drivers and provide actionable recommendations for Q2.

📂 Project Overview
The analysis focuses on answering five core business questions:

Which genres performed best in Q1?

How did each region contribute to total sales?

What weekly sales patterns emerged across the quarter?

Which publishers generated the most revenue?

How do order sizes influence overall sales?

The project includes dashboards, a presentation script, and a full insights summary.

🧹 Data Sources & Preparation
Datasets used:

Monthly sales files (January–March 2024)

Bookstore_Sales.csv

Publisher_Info.csv

Preparation steps:

Unioned monthly datasets

Joined publisher lookup table using PublisherID

Validated fields and corrected minor inconsistencies

Created calculated fields (e.g., order size categories)

Built dashboards for Tasks 1–10

🔍 Key Insights
📚 Genre Performance
History, Sci‑Fi, and Non‑Fiction are the top revenue‑generating genres.

“History, Sci‑Fi, and Non‑Fiction are the strongest‑selling genres across the quarter.”

🌍 Regional Performance
The West region leads in total revenue and cumulative growth.

The South shows the greatest improvement potential.

📅 Weekly Trends
Sales fluctuate naturally week‑to‑week but remain stable overall.

🏢 Publisher Performance
Revenue is highly concentrated, with a small group of publishers generating most sales.

🛒 Order Behavior
Large orders (5+ items) contribute the majority of total revenue.

“Large orders (5+ items) exert major influence on total sales.”
